A new motion model

Introduction

Stumbling around the internet, I discovered an inspirational piece of example code from mickman112 that takes in two mouses as input and visualizes their movement. With a few modifications, this snippet forms the backbone of a new motion model for dual mouse odometry. In the following sections, I'll guide you through this model, which handles the notion that the distance between the two mice is not fixed.
